# Code kata: Prime Factors


## Description

This function breaks down a natural number into its prime factors and returns 
them in ascending order.

__Example__

``` js
1   // => [1]
2   // => [2]
3   // => [3]
8   // => [2,2,2]
24  // => [2,2,2,3]

```

## Run project

__Install__

    npm install

__Run test__

    npm test
